# Upwork 登録・プロフィールガイド

世界最大級のフリーランスプラットフォーム。主に英語圏のクライアント向け。

## 基本情報

- **URL**: https://www.upwork.com
- **言語**: 英語
- **通貨**: USD
- **手数料**: 10%（以前は段階制だったが現在は一律）
- **支払方法**: PayPal、Payoneer、銀行振込、Wise

## 登録フロー

1. アカウント作成（メール or Google/Apple連携）
2. プロフィール作成
3. 本人確認（ID提出）
4. スキルテスト（任意だが推奨）
5. 審査（承認されるとプロフィール公開）

## プロフィール項目

### 必須項目

| 項目 | 文字数制限 | 備考 |
|------|------------|------|
| Title（タイトル） | 70文字 | 検索に影響大 |
| Overview（概要） | 5000文字 | 最初の2-3行が重要 |
| Hourly Rate | - | USD建て |
| Skills | 最大15個 | タグ形式 |
| 職歴 | 各項目2000文字 | Employment History |
| 学歴 | 各項目500文字 | Education |
| ポートフォリオ | 最大20件 | Portfolio Items |

### 推奨項目

- プロフィール写真（顔がはっきり見えるもの）
- Video Introduction（2分以内）
- Languages（対応言語）
- Certifications（資格）
- Skill Tests（Upwork提供のテスト）

## Title（タイトル）の書き方

**形式**: [専門分野] | [強み/特徴] | [キーワード]

**良い例**:
- "AI & Automation Specialist | Python, LLM, Business Process Optimization"
- "Full-Stack Developer | React, Node.js, 10+ Years Experience"
- "Bilingual (EN/JP) Business Consultant | IT Strategy & Digital Transformation"

**避ける**:
- "Freelancer" だけ（具体性なし）
- 長すぎて途切れる
- 絵文字の過剰使用

## Overview（概要）の構成

```
[冒頭フック - 1-2文]
なぜあなたを選ぶべきかを端的に

[専門分野 - 2-3文]
何ができるか、どの分野に強いか

[実績 - 3-4文]  
具体的な数字を含む成果

[仕事の進め方 - 2-3文]
コミュニケーションスタイル、納期厳守等

[CTA - 1文]
「お気軽にご連絡ください」等
```

## 料金設定の目安

| 経験レベル | 時給目安（USD） |
|------------|-----------------|
| エントリー | $15-30 |
| 中級 | $30-60 |
| 上級 | $60-100 |
| エキスパート | $100-200+ |

※分野・専門性により大きく異なる

## 成功のポイント

### Profile Strength

- 「Complete」を目指す
- 全項目を埋める
- スキルテストを受ける

### Job Success Score（JSS）

- 契約完了率を高く保つ
- クライアントとの良好なコミュニケーション
- 納期厳守
- クライアントレビューを依頼

### Connects（応募ポイント）

- 毎月無料で付与
- 追加購入可能
- 質の高い提案に集中投資

## 注意事項

- プラットフォーム外での直接取引は規約違反
- 最初の契約から2年間はUpwork経由が必要
- 虚偽の情報はアカウント停止の原因に
- 複数アカウントは禁止
