# PR文章・可変コンテンツ作成ガイド

## 1. キャッチコピー（一行PR）

**形式**: 15-30文字程度

**パターン例**:
- 「[年数]年の[分野]経験を持つ[職種]」
- 「[強み]で[成果]を実現する[職種]」

**良い例**:
- 「20年のビジネス経験×AI技術で業務改革を支援」
- 「日英バイリンガル対応のITコンサルタント」

## 2. 短文PR（100-200字）

**構成**:
1. 専門分野・強み（1文）
2. 経験・実績の要約（1-2文）
3. 提供価値（1文）

## 3. 長文PR（500-1000字）

**構成**:
1. **冒頭**（2-3文）: 誰で何ができるか
2. **経歴**（3-5文）: 主要な職歴・プロジェクト
3. **強み**（3-5文）: 具体的なスキル・得意分野
4. **実績**（3-5文）: 数字を含む具体的な成果
5. **姿勢**（2-3文）: 仕事への取り組み方
6. **締め**（1-2文）: CTA

## 4. スキルセット

**レベル定義**:
- Expert: 5年以上、他者に教えられる
- Advanced: 3-5年、独力で複雑な課題を解決
- Intermediate: 1-3年、基本業務は独力で可能
- Beginner: 1年未満、学習中

## 作成のコツ

### 数字を入れる
❌ 「多くのプロジェクトに携わりました」
✅ 「50以上のプロジェクトに携わり、90%以上の顧客満足度」

### 具体的な成果
❌ 「業務効率化に貢献しました」
✅ 「RPA導入により月間100時間の工数削減を実現」

### クライアント視点
❌ 「私はPythonが得意です」
✅ 「Pythonでデータ分析を自動化し、意思決定を迅速化します」

## 英語版の注意点

- 能動態で書く（I did X, not X was done）
- 動詞から始める（Developed, Managed, Achieved...）
- 自信を持った表現（I can → I excel at）
