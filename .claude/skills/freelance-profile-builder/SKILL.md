# Freelance Profile Builder

フリーランス求人プラットフォームへの登録準備とプロフィール作成を支援する。

## ワークフロー概要

```
1. 情報収集 → 2. コンテンツ作成 → 3. プラットフォーム別出力
```

## Step 1: 情報収集

### 固定情報（一度設定すれば使い回し可能）
詳細は [references/fixed-info-checklist.md](references/fixed-info-checklist.md) を参照。

### 可変情報（案件や戦略に応じて調整）
- 職種・専門分野
- スキルセット
- 自己PR文（短：100-200字 / 長：500-1000字）
- ポートフォリオ
- 希望時給・単価

## Step 2: コンテンツ作成
